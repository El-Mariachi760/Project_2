# Unbearable
UCB second boot camp group project. 

# Description
Unbearable is web game played by using the arrow keys, and it includes a section where users can add comments, with login of course!

## Built with
* JS
* HTML
* CSS
* Express.js
* Nodejs
* Handlebars
* Sequelize
* Phaser
* Azure
